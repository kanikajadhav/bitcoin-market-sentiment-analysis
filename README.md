# Bitcoin Market Sentiment Analysis

## Overview

This project explores the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data from Hyperliquid.

The objective is to identify patterns between market sentiment and trading outcomes through Exploratory Data Analysis (EDA).

---

## Dataset

The project uses two datasets:

1. Bitcoin Fear & Greed Index
2. Historical Hyperliquid Trader Data

These datasets were provided as part of the internship assignment.

---

## Project Workflow

- Data Cleaning
- Data Preparation
- Dataset Merging
- Exploratory Data Analysis
- Visualization
- Business Insights

---

## Key Findings

- Extreme Greed showed the highest average Closed PnL.
- Extreme Greed also had the highest percentage of profitable trades.
- Fear showed the largest average trade size and trading fees.
- Large differences between mean and median indicate the presence of significant outliers.
